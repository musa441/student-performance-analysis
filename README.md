# Student Performance Analysis

## Overview
This project analyzes student performance data using Python, Pandas, and Matplotlib.

## Features
- Computes student averages
- Assigns letter grades
- Determines pass/fail status
- Filters students based on various conditions
- Finds top and bottom performers
- Calculates descriptive statistics
- Creates visualizations:
  - Histogram
  - Bar chart
  - Pie chart
  - Scatter plot
  - Box plot

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Skills Demonstrated
- Data manipulation
- Statistical analysis
- Data visualization
- Functional programming with `.apply()`
